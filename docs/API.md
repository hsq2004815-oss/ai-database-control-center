# API Reference

The Control Center backend is a read-only API for the independent Personal AI Database Control Center. It proxies the upstream `E:\DataBase` local API when available and falls back to allowlisted local metadata or read-only SQLite queries where implemented.

All project endpoints return:

```json
{
  "ok": true,
  "data": {},
  "error": null,
  "request_id": ""
}
```

Errors return `ok: false` with `error.code`, `error.message`, and `error.details`.

## Endpoints

- `GET /health` - app health, database root status, upstream API status, and endpoint list.
- `GET /domains` - allowed domain list and available operations.
- `GET /domains/{domain}/status` - per-domain file/index status.
- `GET /search` - backend/upstream search with SQLite fallback and local ranking metadata.
- `POST /brief` - proxies the local database `/brief` endpoint and preserves upstream structure.
- `GET /reports` - report metadata from `output/reports`.
- `GET /reports/{domain}/{report_name}` - report markdown preview, filename-only and `.md` only.
- `GET /backend/files` - backend file metadata for `rules`, `topics`, `patterns`, `checklists`, `templates`, `references`, or `reports`.
- `GET /backend/chunks/{chunk_id}` - read one backend chunk from the SQLite index in read-only mode.

## Search

Example:

```powershell
Invoke-RestMethod "http://127.0.0.1:8876/search?domain=backend&q=JWT%20RBAC&limit=5"
```

Common query parameters:

- `domain`: `backend`, another allowed domain, or `all`.
- `q`: search query.
- `limit`: number of results, between 1 and 50.

Search results may include:

- `chunk_id`
- `source_type`
- `title`
- `relative_path`
- `section`
- `content`
- `summary`
- `tags`
- `keywords`
- `priority`
- `trust_level`
- `rank_score`
- `rank_reason`
- `rank_tier`
- `metadata`

The ranking fields are generated by this project after retrieval. They do not require rebuilding upstream indexes.

## Brief

Example:

```powershell
$body = @{
  task = "Build a FastAPI login backend"
  ui_limit = 0
  backend_limit = 8
  workflow_limit = 2
  automation_limit = 0
  assets_limit = 0
} | ConvertTo-Json

Invoke-RestMethod "http://127.0.0.1:8876/brief" -Method Post -ContentType "application/json" -Body $body
```

The `/brief` endpoint is used to retrieve task-specific context from the upstream local knowledge API. Returned fields may include:

- `backend_queries`
- `workflow_queries`
- `ui_queries`
- `automation_queries`
- `asset_queries`
- `backend_chunks`
- `workflow_chunks`
- `ui_chunks`
- `automation_chunks`
- `asset_suggestions`
- `final_handoff`

The upstream structure is preserved so the frontend can show both curated sections and folded debug output.

## Agent Handoff Export

Agent Handoff export is a frontend feature, not a backend endpoint.

- Search exports current ranked results as Markdown.
- Brief exports the current task brief as Markdown.
- Copy and download actions run in the browser.
- No handoff file is written by the backend.
- No handoff file is written to `E:\DataBase`.

## Reports

Examples:

```powershell
Invoke-RestMethod "http://127.0.0.1:8876/reports?domain=backend"
Invoke-RestMethod "http://127.0.0.1:8876/reports/backend/backend-phase-2a-schema-readme-rules-report.md"
```

Report names are filename-only and must end with `.md`.

## Backend Files And Chunks

Examples:

```powershell
Invoke-RestMethod "http://127.0.0.1:8876/backend/files?type=rules"
Invoke-RestMethod "http://127.0.0.1:8876/backend/chunks/backend-rule-auth-and-permission-rules-%E7%A6%81%E6%AD%A2%E5%81%9A%E6%B3%95"
```

`/backend/files` returns metadata only. It does not read full file bodies.

`/backend/chunks/{chunk_id}` reads one indexed backend chunk through SQLite read-only access.

## Error Codes

`UPSTREAM_API_UNAVAILABLE`, `DOMAIN_NOT_FOUND`, `INVALID_DOMAIN`, `INVALID_LIMIT`, `INVALID_QUERY`, `FILE_NOT_FOUND`, `REPORT_NOT_FOUND`, `CHUNK_NOT_FOUND`, `PATH_NOT_ALLOWED`, `DB_NOT_FOUND`, `DB_QUERY_FAILED`, `INTERNAL_ERROR`.

## Safety

- The backend does not provide write, delete, reindex, or migration endpoints.
- `E:\DataBase` and `E:\DataBase\runtime\db` remain outside this repository.
- Search ranking and handoff export do not modify upstream database artifacts.
